# AI Financial Chatbot – BCG GenAI Job Simulation

This project was developed as part of the **BCG GenAI Job Simulation on Forage**.  
It demonstrates how financial data from corporate reports can be analyzed and used to build a simple **AI-powered financial chatbot** that answers queries about company performance.

---

## Project Overview

The project focuses on extracting and analyzing financial data from major companies and using that data to power a rule-based chatbot capable of answering financial questions.

Companies analyzed:
- Microsoft
- Apple
- Tesla

Financial metrics used:
- Total Revenue
- Net Income
- Total Assets
- Total Liabilities
- Operating Cash Flow

The chatbot reads structured financial data and responds to predefined queries about financial performance.

## Features

- Financial data extraction and preprocessing
- Financial trend analysis using **Python and pandas**
- Rule-based chatbot for financial queries
- Interactive command-line chatbot interface
- Query handling for:
  - Highest revenue
  - Lowest revenue
  - Net income comparison
  - Assets and liabilities
  - Operating cash flow

- Python
